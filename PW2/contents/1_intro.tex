\section{intro}

For this lab, a new dataset was chosen, because the ember2018 dataset which comes with scripts for preprocessing, is deprecated.
The scripts are necessary for vectorization of the dataset, but without them we were not able to process the dataset which contains nested JSON Structures.

The newly chosen dataset is the "Malware static and dynamic features VxHeaven and Virus Total" retrieved from the machine learning repository of the UC Irvine. \parencite{UCI2019dataset}

\vspace{1cm}

\item Reload the clean PW1 dataset and splits.

\begin{itemize}
    \item What does your imbalance imply for precision/recall trade-off?
    \item How might imbalance affect model training stability?
\end{itemize}
\item Choose the segmentation method appropriate to your domain
\item Apply at least 3 balancing methods that match your dataset type on the training split only
\end{enumerate}
